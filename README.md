# Musadiq Ahmed - Portfolio

This is a personal portfolio website built with Angular 18 showcasing:
- About (landing page with profile and social links)
- Resume (CV viewer with PDF download)
- Skills (categorized technical skills)

## Quick start (Windows PowerShell)

1. Install dependencies:

```powershell
npm install
```

2. Run the dev server on port 8080:

```powershell
npx ng serve --port 8080 --host 0.0.0.0
```

3. Build for production:

```powershell
npx ng build
```

## Features

- Angular 18 with standalone components
- Angular Material Design UI
- Responsive mobile-first design
- Glass-morphism header with gradient logo
- Gray color scheme (#2c3e50)
- PDF resume viewer
- Social media integration
